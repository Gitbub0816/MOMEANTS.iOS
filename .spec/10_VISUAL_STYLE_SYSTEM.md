# Visual Style System

## Style target

Momeants should feel premium, cinematic, emotional, warm, and memory-focused. The app icon suggests a dark rounded canvas, glowing sunset, photo frame, and heart. The app should inherit that feeling without becoming heavy or visually noisy.

## Core visual principles

- One memory is the hero.
- Use dark cinematic framing around photos.
- Use warm accent light sparingly.
- Avoid flat corporate cards.
- Avoid harsh white screens except forms/settings where clarity matters.
- Avoid gimmicky decorations.
- Use animation as emotional punctuation, not constant motion.

## Typography

Recommended:
- Display/Logo accents: New York, Cormorant Garamond, or a tasteful serif.
- UI/body: SF Pro on iOS / system font.
- Captions: system font with warm spacing.

Rules:
- Do not overuse serif text in the app chrome.
- Captions must be readable over photos.
- Use letter spacing only for small labels.

## Components

### Momeant frame
- Full-screen photo or near-full-screen photo.
- Rounded corners should be moderate, not bubbly.
- Inner shadow or subtle edge glow allowed.
- Photo should feel held, not trapped.

### Gold Heart
- Use for super-like only.
- Animated glow.
- Avoid constant sparkle.
- Should feel rare and meaningful.

### Buttons
- Primary: warm gradient or gold/coral accent.
- Secondary: translucent glass.
- Destructive: muted red, not alarming unless severe.

### Cards
Use cards sparingly. When used:
- Low elevation.
- Thin border.
- Warm surface.
- No chunky SaaS-style card grids.

## App icon design interpretation

The icon communicates:
- dark cinematic environment
- sunset memory
- heart warmth
- framed photo/memory object
- stylized M identity

The app UI should echo this with:
- dark frame surfaces
- warm light accents
- photo-centered layouts
- heart moments reserved for meaningful reactions

## Visual anti-patterns

Do not use:
- TikTok clone styling.
- Instagram clone styling.
- Overly bright neon gradients everywhere.
- Excessive rounded cards.
- Generic dashboard UI.
- Dense analytics-first layouts.
- Cold blue enterprise palette.
