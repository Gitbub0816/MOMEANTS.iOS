# Features and Screens

## Auth and onboarding

Screens:
- Welcome.
- Sign in.
- Create account.
- Verify code.
- Name.
- Username.
- Birthday.
- Avatar.
- Location optional.
- Important People starter.
- Finish.

## Feed

Primary screen. One Momeant at a time.

Elements:
- Photo.
- Author.
- Caption preview.
- Date/captured date.
- Reaction controls.
- Privacy indicator.
- More menu.

## Create Momeant

Screens:
- Select photo.
- Preview/crop.
- Caption.
- Tag Important People.
- Audience.
- Post.

## Memory detail

Includes:
- full photo
- full caption
- comments
- reactions
- people tagged
- date/location
- report/hide options

## Important People

Purpose:
- define meaningful relationships
- improve resurfacing
- improve feed ranking

Features:
- categories
- unlimited people per category
- private notes optional
- linked user if on Momeants
- manual display name if not on app

## Memories tab

Purpose:
- resurfaced memories
- timeline view
- search later
- date-based browsing

## Profile

Includes:
- avatar
- display name
- username
- bio
- meaningful memory count
- privacy controls
- personal Momeants list

## Messaging

MVP: lightweight direct messaging or deferred.

If included:

## Settings

Include:
- Account
- Privacy
